# Breast Cancer Classification using Random Forest

## Project Overview

Project ini merupakan implementasi Machine Learning menggunakan algoritma Random Forest untuk mengidentifikasi faktor risiko kanker payudara berdasarkan dataset kesehatan pasien.

Analisis dilakukan mulai dari pemahaman dataset, data cleaning, exploratory data analysis (EDA), hingga pemodelan machine learning menggunakan Python.

---

## Tujuan Project

Tujuan dari project ini adalah:

* Mengidentifikasi faktor-faktor yang mempengaruhi risiko kanker payudara
* Melakukan analisis statistik dan eksplorasi data
* Membangun model prediksi menggunakan Random Forest
* Mengevaluasi performa model menggunakan Cross Validation

---

## Dataset

Sumber dataset:
https://data.mendeley.com/datasets/7jhddnpz2p/1

Dataset berisi berbagai variabel seperti:

* Age
* Menarche age
* Breastfeeding duration
* Biopsies
* BMI (IMC)
* Alcohol consumption
* Tobacco use
* Exercise
* Emotional condition
* Depressive status
* Hyperplasia
* Cancer (target variable)

Dataset memiliki sekitar **1686 data pasien** dengan dua kelas:

* Cancer (Yes)
* Non-Cancer (No)
